# Practice Area Copy

Cria páginas de áreas de atuação que convertem, evitando o erro mais comum: listas genéricas de serviços.

## Estrutura da Página

### 1. Introdução (Liderar com o problema do cliente)

NÃO comece com:
> "O escritório XYZ é referência em direito de família..."

COMEÇE com:
> "Passar por um divórcio é um dos momentos mais difíceis da vida. Além da dor emocional, surgem medos: e meus filhos? E meus bens? Como recomeçar?"

### 2. Subáreas (valores, não só nomes)

| Em vez de... | Escreva... |
|--------------|------------|
| Divórcio consensual | "Quando ambos querem o divórcio: um caminho mais rápido e tranquilo" |
| Divórcio litigioso | "Quando não há acordo: sua defesa para proteger seus direitos" |
| Guarda de filhos | "O que é melhor para seus filhos? Nós ajudamos a construir um acordo justo" |
| Pensão alimentícia | "Segurança financeira para quem mais importa" |

### 3. Como funciona o processo (Reduz ansiedade)

Explique em 3-4 passos simples:

```
1. ✅ Primeiro contato: você conversa conosco, sem compromisso
2. 📋 Análise do caso: entendemos sua situação e explicamos os caminhos
3. ⚖️ Estratégia: definimos juntos a melhor abordagem
4. 🤝 Acompanhamento: você não fica perdido em nenhuma etapa
```

### 4. Perguntas frequentes (FAQ)

Responda o que todo cliente pergunta:

- Quanto custa?
- Quanto tempo demora?
- Preciso ir ao tribunal?
- Como escolher o melhor advogado?

### 5. Call to Action

Inclua CTA após cada seção importante.

## Template de Conteúdo

Use este template para cada área de atuação:

```markdown
## [Título: Benefício + Área]

[Parágrafo de abertura: reconheça a dor do cliente]

### Como podemos ajudar

[Liste subáreas com abordagem de valor, não só nomes técnicos]

### O processo é simples

[Passo a passo em bullets]

### Perguntas frequentes sobre [área]

[FAQ com 3-5 perguntas]

### Pronto para conversar?

[CTA final]
```
