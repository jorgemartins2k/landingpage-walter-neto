# SEO Content Optimizer

Equilibra SEO e experiência do usuário, criando conteúdo que ranqueia e converte.

## Princípios Fundamentais

1. **Escreva para pessoas primeiro, Google depois** 
2. **Use palavras-chave naturalmente** — não force
3. **Estruture para escaneabilidade** (bullets, subtítulos)
4. **Mantenha conteúdo atualizado** — dados desatualizados quebram confiança 

## Onde Inserir Palavras-Chave

| Local | Importância | Exemplo |
|-------|-------------|---------|
| Título (H1) | Alta | "Advogado de Divórcio em São Paulo: Atendimento Humanizado" |
| Subtítulos (H2, H3) | Média | "Como funciona o processo de divórcio?" |
| Primeiro parágrafo | Alta | Responda a pergunta principal imediatamente |
| Meta description | Alta | Resumo com benefício e CTA |
| URL | Média | `/divorcio/sao-paulo/` |
| Imagens (alt text) | Baixa | "advogada-atendendo-cliente-divorcio" |

## Estrutura de Conteúdo SEO-Friendly

```markdown
# [Título com palavra-chave principal]

[Parágrafo de abertura respondendo à pergunta principal do usuário]

## [Subtítulo H2 com variação da keyword]

[Conteúdo em blocos curtos]

### [Subtítulo H3 com especificidade]

[Listas, bullets, tabelas para escaneabilidade]

## Perguntas frequentes sobre [tema]

### O que é [termo]?
[Resposta clara e objetiva]

### Quanto custa [serviço]?
[Informação de valor]

### Como funciona [processo]?
[Passo a passo]

## Pronto para resolver seu caso?

[CTA final]
```

## Dicas de SEO Jurídico

- Use **schema LocalBusiness** para escritórios com localização física 
- Crie **páginas localizadas** para cidades de atuação 
- Inclua **links internos** entre áreas de atuação e blog 
- Atribua conteúdo a **advogados reais** (aumenta credibilidade) 
- Atualize **datas e referências legais** regularmente
