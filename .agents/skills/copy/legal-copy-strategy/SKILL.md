# Legal Copy Strategy

Este skill define a estratégia de copywriting para sites de advogados antes da criação de conteúdo.

## Árvore de Decisão

1. **Identificar o nicho do escritório**
   - Família / Divórcio → tom empático e acolhedor
   - Criminal → tom firme, de defesa e proteção
   - Trabalhista → tom de justiça e reparação
   - Previdenciário → tom de cuidado e direito adquirido
   - Empresarial → tom profissional e estratégico
   - Sucessões → tom sensível e planejamento

2. **Definir a persona do cliente ideal**
   - Quem é? (idade, profissão, renda)
   - Qual é o problema principal?
   - Qual o medo/obstáculo?
   - Qual a motivação para contratar?

3. **Estabelecer o Tom de Voz (TOV)**
   - Escolha 3 adjetivos que definem o tom (ex: acolhedor, confiante, claro)
   - Defina o que EVITAR (jargão jurídico, linguagem agressiva)

## Template de Briefing

Use este template para extrair informações:

```
## Briefing de Copy - [Nome do Escritório]

### 1. Nicho e Áreas de Atuação
- Principal: 
- Secundárias:

### 2. Persona
- Quem é o cliente ideal?
- Quais dores ele sente?
- O que ele busca ao chegar no site?

### 3. Diferenciais (USP)
- O que torna este escritório único?
- [ ] Abordagem humanizada
- [ ] Tecnologia/AI
- [ ] Mais de X anos de experiência
- [ ] Atendimento 24h
- [ ] Localização privilegiada
- [ ] Outro: ________

### 4. Tom de Voz
- Palavras que definem: [ex: acolhedor, profissional, acessível]
- Palavras a evitar: [ex: "lide", "parte autora", "vítima"]

### 5. Trust Markers disponíveis
- [ ] Avaliações Google (nota: __)
- [ ] Selos/Prêmios (quais?)
- [ ] Casos de sucesso
- [ ] Depoimentos
- [ ] Tempo de mercado
- [ ] Associações (OAB, comissões)
```

## Saída Esperada

Ao final, o skill deve gerar um documento de estratégia que servirá de base para todos os outros skills de copy.
